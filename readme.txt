== Description ==

Simple plugin that adds an All Settings submenu to the Settings menu for Administrators and above, enabling one-click access to the options.php file.

== Installation ==

1. Upload `all-settings.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What exactly does this do? =

This plugin adds an All Settings submenu to the Settings menu, providing one-click access for Administrators and above to the WordPress options.php file. This file allows editing of some of the values in the wp-options table.
